### Requirements

- You need to install both [Node.js](https://nodejs.org/en/download/) and [Yarn](https://yarnpkg.com/) to run this project.

```bash
# Install all dependencies
$ yarn

# Run the api 
$ yarn server

# Run the web server
$ yarn dev
```

The app will be at `http://localhost:3000`
